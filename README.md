# Mastermind Game

This is a browser-based implementation of the classic **code-breaking logic game**. Inspired by the original board game, players must guess a hidden sequence using logical deduction within a limited number of attempts.

## How to Access?

Open the link in broswer: https://alinaaaw.github.io/mastermind/

## How to Play?

### Objective

The goal is to guess the **hidden sequence** of colors within a **limited number** of attempts based on your choice of difficulty. Use the **feedback** for each guess wisely to figure out the correct combination before your guesses run out.

> [!TIP]
> * Start with distinct colors to maximize information  
> * Focus on logical deduction and elimination rather than guessing randomly

### Difficulty Level
Select a difficulty to match your desired challenge:  
* **Easy**: 20 guesses in total
* **Medium**: 12 guesses in total
* **Hard**: 5 guesses in total

### What the Feedback Means

After each time user click **submit guess**, the game provides hints:  

* **Black** circles: **correct** color in **correct** position  
* **White** circles: **correct** color in **wrong** position  

Use this information to refine your guesses strategically.  

> [!IMPORTANT]
> The hint circles do not indicate specific positions. They only provide hint on the number of correct color and/or position.

### Resetting the Game

Press "Reset Game" to start the game all over from choosing difficulty level
